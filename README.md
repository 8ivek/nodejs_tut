# Nodejs for Beginners

Beginners guide to Nodejs.

## How to use
_node filename.extension_

### example:

> __node index.js__

## NPM

Go inside folder specific folders and execute:
> __npm install__

*wherever there is package.json (or package-lock.json) file*.
